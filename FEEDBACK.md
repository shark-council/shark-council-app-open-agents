# Uniswap API and Developer Platform Feedback

Here is our builder experience and feedback regarding the Uniswap API and Developer Platform.

## 1. Skill Installation Command Issue

The official command to get the skill (`npx skills add uniswap/uniswap-trading --skill swap-integration`) from the https://developers.uniswap.org/docs/api-reference/aggregator_quote doesn't work, even after GitHub authentication. We were only able to install the skill using the command from [skills.sh](https://skills.sh/uniswap/uniswap-ai/swap-integration).

## 2. Missing Integrator Fee Information in Skill

The installed skill does not include information about the integrator fee. We had to read the official documentation to find out how to implement it.

## 3. Feature Request - On-chain Advanced Orders

It would be highly beneficial to have on-chain features implemented to support placing take profit and stop loss orders directly.
